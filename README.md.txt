# Project HTML Motor

Project ini dibuat untuk memenuhi tugas praktikum HTML.

Website ini menampilkan informasi sederhana tentang motor Honda CBR250RR
dengan menggunakan HTML dan sedikit CSS untuk memperindah tampilan.

## Fitur
- Menampilkan judul dan deskripsi
- Menampilkan gambar motor
- Tampilan sederhana dan menarik

## Teknologi
- HTML
- CSS
- Git & GitHub
